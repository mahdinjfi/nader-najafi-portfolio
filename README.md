# وب‌سایت پرتفولیو نادر نجفی

وب‌سایت فارسی، راست‌چین، مدرن و حرفه‌ای برای معرفی خدمات نادر نجفی در حوزه نصب اسلب، کاشی، سنگ، پرسلان، کانتر و روشویی.

## تکنولوژی‌ها

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- طراحی کاملاً RTL و واکنش‌گرا

## اجرا

```bash
npm install
npm run dev
```

سپس آدرس زیر را باز کنید:

```text
http://localhost:3000
```

## ساختار پروژه

```text
app/

  layout.tsx
  page.tsx
components/
  About.tsx
  Contact.tsx

  InternationalCollaboration.tsx

  Stats.tsx

public/
  images/
    README.md
```

## تصاویر واقعی

فعلاً سایت از placeholderهای شیک و مرتبط با سنگ و اسلب استفاده می‌کند. برای نسخه نهایی، تصاویر پروژه‌ها را داخل `public/images` قرار دهید و جایگاه‌های مشخص‌شده در کامپوننت‌ها را با `next/image` جایگزین کنید.
